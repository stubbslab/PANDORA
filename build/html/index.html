Overview¶

Welcome to the PANDORA project documentation. PANDORA is a versatile instrument control system that orchestrates multiple laboratory subsystems—such as the monochromator, shutter, flip mounts, Keysight electrometers, and Zaber stages— to provide a unified, automated workflow. This documentation offers a high-level introduction, installation guidance, quick-start examples, and detailed API references.

How to Install¶

Installation instructions are under development. Please check back soon for a comprehensive guide on setup, configuration, and dependency management.

Quick Start¶

Kickstart your experience with PANDORA using this minimal example:

from pandora_controller import PandoraBox
pandora = PandoraBox()
pandora.initialize_subsystems()

Hello, world!

Upon initialization, the subsystems assume their default safe states:

  • Monochromator: go to home state (400 nm).

  • Shutter: Closed, blocking the light path.

  • Flip Mounts: Not on the optical path.

  • Keysight Electrometers: On standby.

  • Zaber Stages: At home (off light path).

Once the subsystems are initialized, you can control them individually or in combination to suit your experimental needs.

For more details on controlling the hardware, see the Controlling Subsystems.
